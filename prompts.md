# Мои промпты для ИИ

## Шаблон для ИИ-копирайтинга

**Роль:** Ты профессиональный SMM-маркетолог.

**Задача:** Напиши вовлекающий пост для соцсетей.

**Контекст:** 
* [Сюда вставлять тему бизнеса и суть публикации]

**Ограничения:**
* Текст до 500 символов.
* Живой, разговорный язык без сложных канцеляризмов.
* Добавить 3 релевантных хэштега в конце.
* Использовать эмодзи в качестве визуальных акцентов.
